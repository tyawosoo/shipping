# Shipping

这个仓库现在提供的是“最优发货方案”静态网页版本，保留了原来的 `shipping_app.py` 和价格 Excel 作为参考数据来源，但对外分享推荐直接使用 GitHub Pages 页面。

目标访问地址：

- `https://tyawosoo.github.io/shipping/`

## 页面功能

- 选择目的地省份和城市
- 输入 A 类与 B 类发货数量
- 自动计算最优发货方案
- 同时比较温控箱、整车、混合发货三类方案
- 按总成本排序显示前 5 个推荐方案
- 支持一键复制结果，便于转发

## 说明

静态页面复用了原 Streamlit 版本的核心计费逻辑：

- 温控箱方案：按不同箱型容量和线路价格组合求解
- 整车方案：按重量区间价格和最低收费计算
- 混合方案：按预设比例拆分箱发与整车发

这样部署后不再依赖 Streamlit Community Cloud，也不会因为空闲而进入休眠。
